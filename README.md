# WeTube

WeTube는 실서비스인 유튜브를 클론한 프로젝트입니다.<br>
로그인/소셜로그인, 로그아웃, 유저프로필, 동영상 업로드, 녹화 및 다운로드, 댓글, 좋아요 등의 기능이 있습니다.<br><br>
:star: 아래 데모 사이트 주소로 직접 확인해보세요 :star:<br>
